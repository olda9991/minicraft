How this works: CheerpJ is a Java-to-JavaScript runtime that executes standard JAR files in the browser with zero code changes. The MiniCraft.jar here is the exact same JAR built from src/MiniCraft.java via javac.

Controls: WASD to move, Space to jump, Click to break/place, 1-9 hotbar, F for Survival/Creative, F9 for 3D mode, F10 for VR SBS, Esc for menu.

Note: Browser audio requires a user click first. For multiplayer / voice chat, use the desktop ./run.sh.
