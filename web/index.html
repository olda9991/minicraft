MiniCraft Web

v6.4.2 — Original Java code running in browser via CheerpJ (zero rewrites)

Click Launch to start MiniCraft in your browser

This loads the original MiniCraft.jar JAR via CheerpJ

How this works: CheerpJ is a Java-to-JavaScript runtime that executes standard JAR files directly in the browser. The MiniCraft.jar you see here is the exact same JAR built from src/MiniCraft.java — zero code changes, zero rewrites.

Controls: WASD to move, Space to jump, Click to break/place, 1-9 hotbar, F for Survival/Creative, F9 for 3D mode, F10 for VR SBS, Esc for menu.

Server: Use python3 -m http.server 8765 in the web/ folder, then open http://localhost:8765.

Note: CheerpJ supports Swing but audio/networking may have limitations in browser. For offline play or better performance, use the desktop ./run.sh.
