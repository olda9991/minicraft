# MiniCraft - Windows Setup Guide

## Step 1: Install Java 21
1. Go to https://adoptium.net/
2. Click **"Download Latest LTS for Windows"**
3. Run the installer (`.msi` file)
4. Click **Next** → **Install** → **Finish**

## Step 2: Download MiniCraft
1. Ask your friend (the host) for the web status page URL
2. Open that URL in your browser
3. Click **"Download for Windows"** button
4. Extract the downloaded `MiniCraft.zip` somewhere (like `Desktop/MiniCraft`)
5. Right Click on PixelPurl.ttf and click on a button saying install font you need it for game to show text
## Step 3: Run the Game
1. Open the extracted folder
2. Double-click **`run.bat`**
3. Wait for the game window to appear
4. Click **Multiplayer** → **Direct Connect**
5. Type the server address your friend gave you (looks like `bore.pub:53082`)
6. Click **Connect**

## Step 4: Play!
- WASD / Arrow keys = Move
- Left Click = Break blocks
- Right Click = Place blocks
- Scroll wheel = Change selected block
- E = Crafting menu
- F = Survival/Creative mode
- T = Chat
- ESC = Save & Exit

## Troubleshooting

**"Java not found" error:**
→ Install Java 21 from https://adoptium.net/

**"Cannot connect" error:**
→ Make sure the host's server is running
→ Check the server address is typed correctly (copy-paste it!)
→ Try `localhost:25565` if testing on the same PC

**Weird font or missing textures:**
→ Make sure you extracted ALL files from the zip
→ Don't move files around inside the MiniCraft folder

## Need help?
Ask the server host to refresh their web status page for the current connection info.
